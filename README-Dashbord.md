# Dashboard

Kryptalis is built around a modern, fast and intuitive user interface.

The goal is to make infrastructure management accessible to everyone.

---

## Modern Interface

Built with:

* Next.js 15
* React 19
* TypeScript
* Tailwind CSS
* shadcn/ui

Features:

* Dark Mode
* Light Mode
* Responsive Design
* Real-time Updates
* Keyboard Shortcuts

---

## Overview Dashboard

A global view of your infrastructure.

Monitor:

* Servers
* Applications
* Domains
* Databases
* Emails
* Containers
* Resources

From a single screen.

---

## Projects

Each project centralizes:

* Applications
* Domains
* SSL Certificates
* Environment Variables
* Databases
* File Storage
* Monitoring
* Backups

---

## Real-Time Monitoring

Live metrics:

* CPU Usage
* Memory Usage
* Disk Usage
* Network Activity
* Container Statistics

Without page refresh.

---

## Deployment Interface

Deploy applications visually.

Features:

* Git Repository Connection
* Branch Selection
* Environment Variables
* Build Configuration
* Deployment History
* Rollbacks

---

## File Manager

Integrated web file explorer.

Features:

* Upload Files
* Download Files
* Edit Files
* Create Directories
* Compress Files
* Extract Archives

No FTP client required.
